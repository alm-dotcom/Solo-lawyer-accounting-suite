# Solo-lawyer-accounting-suite

A comprehensive practice management system designed specifically for solo practitioners and small law firms.

## Features

- **Client & Matter Management**: Track clients and associated legal matters
- **Time Tracking**: Built-in timer for accurate billable hours tracking
- **Trust Account Management**: RPC 1.15A compliant trust accounting
- **Operating Account**: Track firm expenses and revenue
- **Invoice Generation**: Create professional invoices with automatic calculations
- **Task Management**: Organize deadlines and assignments
- **Calendar**: Integrated calendar for scheduling
- **Budget Tracking**: Monitor client budgets and expenses
- **Scrivener Integration**: Export/import data for document writing

## Design

Professional interface featuring:
- Navy blue and gold color scheme
- Clean, modern dashboard cards
- Responsive layout
- Professional typography
- Elegant glassmorphism effects

## Technology Stack

- **Frontend**: HTML5, TailwindCSS, Vanilla JavaScript
- **Backend**: Node.js, Express.js (optional local backend)
- **Database**: SQLite (local storage)
- **Data Format**: JSON for import/export

## Getting Started

### Simple Usage (HTML Only)
1. Open `Solo Lawyer Accounting Suite.html` in a modern web browser
2. All data is stored in browser localStorage

### With Backend (Optional)
1. Navigate to the `backend` folder
2. Run `npm install`
3. Run `npm start`
4. Backend API runs on http://localhost:3001

## Customization

- Firm branding can be updated in Settings
- Logo image: `lamar-allen-law-firm-logo.png`
- Color scheme customizable via CSS variables

## Legal Compliance

- RPC 1.15A trust account compliance built-in
- Audit trail for all transactions
- Separate trust and operating accounts
